Live Bidding App

  • Category: API
  • Reference: Documentation

An api server to facilitate online bidding of products was developed using Node Js and MySQL .The routes were categorized into admin,user and authentication routes (with 2 factor authentication, and google sign integrated).The Api server was hosted in heroku. The product and admin routes can be accessed by the admin to add and update product details. The user can bid the product and based on the maximum price at that time the admin decides to which user the product can be sold.
